# What you doin'?

## Installation

### Requirements
* Node.js 8.9+

### How to install
* Clone this repo: ```git clone https://github.com/ctrlaltdev/WYD.git```
* Move inside the directory: ```cd WYD```
* Install the npm packages: ```npm i``` or ```yarn install```
* Do stuff
* Pet a cat
* Enjoy life
