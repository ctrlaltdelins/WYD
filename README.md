# What you doin'?

## Installation

### Requirements
* Sqlite3

#### To compile from scratch
* Make
* Sqlite3-libs

### How to install
* Clone this repo: ```git clone https://github.com/ctrlaltdev/WYD.git```
* Move inside the directory: ```cd WYD```
* Create the executable: ```make``` then ```make install```
* Do stuff
* Pet a cat
* Enjoy life
